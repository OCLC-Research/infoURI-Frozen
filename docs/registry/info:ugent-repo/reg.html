identifier-syntax
description 
Identifiers in the ugent-repo namespace are used to
identify information assets used in the Los Alamos aDORe
repository at the University Library Ghent. The syntax is of
the form:

     info:ugent-repo/_type_/_identifier_

whereby:

     _type_  gives an indication of the nature and
             purpose of the identifier. Current values
             for _type_ include 'i' , 'opac'. Their
             meaning is explained in the Namespace
             Information section of the info:lanl-repo
             namespace.

     _identifier_  is a unique string assigned to the
             information asset.

Examples:
*    info:ugent-repo/i/000b8c20-51b3-11d8-94f8-e15f3b151c3cffff
*    info:ugent-repo/opac/rug01-0219201
uri
identifier-normalization
description 
All identifiers in the ugent-repo namespace are case-sensitive.
