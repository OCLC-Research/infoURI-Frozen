description 
DDBJ/EMBL/GenBank Accession Numbers are identifiers assigned to all publicly
available nucleotide sequences submitted to the International Nucleotide
Sequence Database DDBJ/EMBL/GenBank.
Nucleotide sequence accession numbers can be 1 letter + 5 digits or 2 letters
+ 6 digits.
Whole Genome Shotgun sequence project accession numbers are 4 letters + 8-10
digits.  The first two digits represent the version number of the assembly
and the subsequent 6-8 represent the sequence contig number.
