identifier-syntax
description 
The ncid identifier is a string of length 10, with the 
following composition:

(*) The first two characters must be alphabetic.  The 
following 4 patterns are supported:

BA  or  BN  assigned for books 
AA  or  AN  assigned for serials

(*) The next 7 characters must be digits (0-9).

(*) The last  character is a check sum (0-9,X).

namespace-information
description 
the ncid identifier is a sequentially assigned number that 
uniquely identifies an NACSIS-CAT bibliographic record.  In 
November 2006 there were about 8 million bibliographic and 
90 million holding records in NACSIS-CAT. 

If a record is merged into another or if a record is deleted, 
its number is discontinued and never re-used. 

Information on NACSIS-CAT is available at 
http://www.nii.ac.jp/CAT-ILL/contents-e/e_ncat_top.html.
