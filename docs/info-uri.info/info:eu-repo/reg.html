identifier-syntax
description 
Identifiers in the eu-repo namespace are used to identify 
information assets used in the European Research Libraries. 
The syntax is of the form:
  info:eu-repo/_type_/_identifier_

whereby:
* _type_ gives an indication of the nature and purpose of 
the identifier.
Current values for _type_ include: 'semantics' and 'dai'.  The meaning 
is explained in the Namespace Information section.
* _identifier_ is a unique string assigned to the 
information asset.
* _identifier_ can be replaced by _subType_/_identifier_

Examples:
* info:eu-repo/semantics/descriptiveMetadata
* info:eu-repo/semantics/objectFile
* info:eu-repo/semantics/humanStartPage
* info:eu-repo/dai/nl/12345
* info:eu-repo/semantics/researchProposal

namespace-information
description 
Identifiers in the eu-repo namespace are used to identify 
information assets used in the European Research Libraries.  
Different types of identifiers are used in the repository,
including: Semantic  Identifiers, Publication Types and 
Digital Author Identifiers.

# Identifiers for Semantic terms: Identifiers used to 
represent properties of assets stored in the repository, or 
relationships between such assets.  These identifiers are 
used in RDF or DIDL statements.  These identifiers are of 
the form 
info:eu-repo/semantics/_identifier_ 
whereby _identifier_ is a unique string acorded to a 
property or type of relationship.
Official Enumerations: 
* info:eu-repo/semantics/descriptiveMetadata
--> Defines the information about the item
* info:eu-repo/semantics/objectFile
--> Defines the object item
* info:eu-repo/semantics/humanStartPage
--> defines the location used for human beings to start

# Identifiers for publication types: Identifiers to 
represent the type of publication. These identifiers are of 
the form info:eu-repo/semantics/_identifier_
Official Enumerations:
* info:eu-repo/semantics/annotation
* info:eu-repo/semantics/article
* info:eu-repo/semantics/bachelorThesis
* info:eu-repo/semantics/book
* info:eu-repo/semantics/bookPart
* info:eu-repo/semantics/bookReview
* info:eu-repo/semantics/contributionToPeriodical
* info:eu-repo/semantics/doctoralThesis
* info:eu-repo/semantics/researchProposal
* info:eu-repo/semantics/lecture
* info:eu-repo/semantics/masterThesis
* info:eu-repo/semantics/researchPaper
* info:eu-repo/semantics/patent
* info:eu-repo/semantics/preprint
* info:eu-repo/semantics/report
* info:eu-repo/semantics/studentThesis
* info:eu-repo/semantics/technicalDocumentation

# Identifiers for Authors: Identifiers that are used 
to represent the author in a unique way.
* info:eu-repo/dai/_ISO3166_/_uniqueNationalString_
Where by: _ISO3166_ is the two letter code for countries.
http://www.iso.org/iso/country_codes/iso_3166_code_lists/
Example: 
* info:eu-repo/dai/nl/12345
