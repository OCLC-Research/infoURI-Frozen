description 
RefSeq Accession Numbers are identifiers assigned to all publicly available
nucleotide sequences provided by the NCBI RefSeq project. Nucleotide sequence
accession numbers can be 2 letter + underscore ("_") + 6-8 digits. The first
two letters impart information about the type of sequence data contained
within the record. RefSeq records that are based on Whole Genome Shotgun
sequence project data may have accession numbers that are 2 letter +
underscore + 4 letters + 8-10 digits. The first two digits represent the
version number of the assembly and the subsequent 6-8 represent the sequence
contig number.
