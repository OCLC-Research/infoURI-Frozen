identifier-syntax
description 
Identifiers in the arxiv namespace are used to identify 'papers' in the 
arXiv.org e-print repository. The syntax of the identifier is of the form:

  info:arxiv/<internal_id>

where:

  <internal_id> is the arXiv.org paper identifier
  <internal_id> = <archive> '/' <papernum>

  <archive> is the archive name with the repository
  <archive> = <lcletter> *( <lcletter> | '-' )
 
  <papernum> is the number of a given paper within the archive
  <papernum> = 7*<digit>

  <lcletter> = 'a' - 'z'
  <digit> = '0' - '9'

At present, <papernum> is always 7 digits but this is likely to be 
increased in the near future. 

A 'paper' in arXiv may include one or more versions of an article or may have 
been deleted. Each version may be composed a number of files in different 
formats and a number of disseminations may also be available. As with historical
use of arXiv's internal identifiers, info: identifiers using the format above 
identify the compound digital object including all versions.

It is intended to extend this identifier syntax to identify versions and 
components of papers.

identifier-normalization
description 
Identifiers in this namespace are case sensitive. At present this means that all
letters will be lower case.
