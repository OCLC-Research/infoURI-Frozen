identifier-syntax
description 
Each identifier starts with <info:nla/collection-id>, where the <collection-id> represents one of the following sub-namespaces. The syntax of the remainder of the identifier following the sub-namespace varies according to the type of object. 

Manuscripts: /ms-<ms collection no.>-<series no.>-<item no.>-<sequence no.>-<tile no.>-<role code> 
Example: 
info:nla/ms-ms9-1-s1-e
which is currently resolvable as:
http://nla.gov.au/nla.ms-ms9-1-s1-e (examination copy of the first page of the first item in series 1 of the papers of Sir Joseph Banks.) 



Oral History: /oh-<trc collection no.>-<series no.>-<item no.>-<part no.>-<sequence no.>-<piece no.>-<fragment no.>-<role code> 

Example: 
info:nla/oh-0234-0003-0001-m
which is currently resolvable as:
http://nla.gov.au/nla.oh-0234-0003-0001-m (master recording of the first tape in series 3 of TRC0234) 

Maps: /map-<item no.>-<part no.>-<sequence no.>-<tile no.>-<subTile no.>-<role code> 
Example: 
info:nla/map-rm2099-t
which is currently resolvable as:
http://nla.gov.au/nla.map-rm2099-t (a thumbnail copy of rare map number 2099) 

Pictures: /pic-<item no.>-<part no.>-<sequence no.>-<role code> 

Example: 
info:nla/nla.pic-an6729917-t
which is currently resolvable as:
http://nla.gov.au/nla.pic-an6729917-t (a thumbnail copy of a single picture from the Ellis Rowan collection) 

Music: /mus-<item no.>-<sequence no.>-<role code> 

Example: 
info:nla/nla.mus-an7579855-s2 
which is currently resolvable as:
http://nla.gov.au/nla.mus-an7579855-s2 (an unnumbered title page that is the second image in the group representing music item 7579855) 

Monographs (gen=General Collection; aus=Australian Collection): /gen-<work no.>-<chapter no.>-<section no.>-<sequence no.>-<role code> .aus-<work no.>-<chapter no.>-<section no.>-<sequence no.>-<role code> 

Example: 
info:nla/nla.gen-isbn0590353403-2-s1-v
which is currently resolvable as:
http://nla.gov.au/nla.gen-isbn0590353403-2-s1-v (a view copy of the first page of chapter 2 the book with this ISBN) 


Serials (gen=General Collection; aus=Australian Collection): /gen-<serial identifier>-<issue vol.>-<issue no.>-<issue part>-<issue date>-<subUnit no.>-<sequence no.>-<role code> .aus-<serial identifier>-<issue vol.>-<issue no.>-<issue part>-<issue date>-<subUnit no.>-<sequence no.>-<role code> 

Example: 
info:nla/nla.aus-issn10399992-s1-v
which is currently resolvable as:
http://nla.gov.au/nla.aus-issn10399992-s1-v (a view copy of the first page of the serial with this ISSN)

namespace-information
description 
To identify a born-digital or a digitised object in the managed digital collections of the National Library of Australia, an intelligent identifier has been constructed which utilises a sub-namespace to encode the collection type and, based on the type, rules for accessing component parts of the object.
